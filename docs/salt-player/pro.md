# 椒盐音乐 Pro

为了椒盐音乐的持续发展、为用户提供一些特殊功能，椒盐音乐设计了**椒盐音乐 Pro** 。

椒盐音乐 Pro 位于椒盐音乐 - 设置 - 椒盐音乐 Pro 。

开启椒盐音乐 Pro 后可以使用页面下方提供的功能（后续可能在其他页面也会有显示）。
