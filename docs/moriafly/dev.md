# 产品路线图

经过 2022 年 4 月份调整，Moriafly 软件产品大版本更新会进行内部组讨论测试确定预发布功能和规划（不对外公布），经过规划后进行公开两轮测试，面向 QQ 群聊和 Telegram 频道（主要为 Telegram 频道）进行。

以下是产品发布路线图，时间仅为预计时间，最终发布时间可能有变化。
